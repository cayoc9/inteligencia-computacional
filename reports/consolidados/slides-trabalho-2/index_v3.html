Trade-off

O melhor resultado e um ponto operacional, nao uma verdade clinica pronta

O ganho em sensibilidade so e honesto quando o custo em falsos positivos aparece na mesma tela.

Ponto escolhido para defesa
METABRIC ensemble chegou a 1 falso negativo, mas aceitando 146 falsos positivos.
Interpretacaosensibilidade muito alta
Riscosobrecarga operacional
O que a banca pode cobrar
1validacao externa antes de qualquer uso clinico
2analise de custo para o volume de FP
3survival analysis para horizonte temporal explicito
Trade-off assumido, nao escondido12 / 16
4. Conclusao

O ganho principal foi metodologico e informacional

1Remover vazamento tornou o experimento defensavel.
2METABRIC trouxe sinais clinicos mais ricos.
3Ensemble ajudou como ponto de alta sensibilidade.
4Neuro-fuzzy ensinou como comparativo, nao como campeao.
Conclusao: complexidade sem dados bons nao resolve o problema13 / 16
