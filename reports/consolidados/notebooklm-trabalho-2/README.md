## Regra de uso

- Use os arquivos `01` a `06` como fontes.
- Use o arquivo `07` para copiar prompts no Studio/Audio Overview do NotebookLM.
- Se quiser gerar audios curtos e tematicos, siga a numeracao dos subtopicos no arquivo `07`.
