# Stack do Projeto

## Linguagem e Ambiente
- Python 3.12.3 em ambiente virtual local `.venv`.
- O diretório atual não responde como repositório git funcional via `git rev-parse`.

## Bibliotecas Observadas
- Dados e EDA: `pandas`, `numpy`, `matplotlib`, `seaborn`.
- Machine Learning clássico: `scikit-learn`, `joblib`.
- Redes neurais: `tensorflow`, `keras`.
- Ambiente interativo disponível: `jupyterlab`, `notebook`, `ipykernel`, `ipython`.

## Artefatos
- Dataset principal: `data/sleep_health_dataset.csv` com 100.001 linhas incluindo cabeçalho.
- Figuras: `reports/figures/`.
- Modelos e históricos: `models/`, `models/v2/`, `models/optimized/`.

## Lacunas de Reprodutibilidade
- Não há `requirements.txt`, `pyproject.toml` ou `environment.yml`.
- Não há notebooks `.ipynb` no projeto, apesar de Jupyter estar instalado no ambiente.
