# Checklist: Revisão de Gráficos

> Use antes de colocar um gráfico em qualquer apresentação ou relatório.

## Princípio 1: Contexto

- [ ] O tipo de gráfico é adequado ao público? (técnico vs não-técnico)
- [ ] O gráfico responde à pergunta que o público precisa responder?
- [ ] A mensagem principal está clara sem explicação adicional?
- [ ] O nível de detalhe é adequado ao mecanismo (ao vivo/documento)?

## Princípio 2: Visual Eficaz

- [ ] O tipo de gráfico é o mais adequado para esses dados?
- [ ] Gráfico de barras: linha de base começa em ZERO?
- [ ] Se for pizza/rosca/3D: **remova imediatamente**
- [ ] Se for eixo Y secundário: **substitua** (legende direto ou separe em 2)
- [ ] Categorias estão ordenadas logicamente?
- [ ] Dados categóricos com nomes longos → barras horizontais?
- [ ] Série temporal → gráfico de linhas (intervalos consistentes)?

## Princípio 3: Saturação

- [ ] Todas as bordas e linhas têm função?
- [ ] Sem efeitos decorativos (sombra, 3D, gradiente, brilho)?
- [ ] Cores são usadas com propósito? (não decorativas)
- [ ] Eixo e legendas de dados não são redundantes? (escolha um)
- [ ] Espaço em branco presente e usado para organizar?
- [ ] Se eu remover este elemento, a informação se perde? Se não, remova.

## Princípio 4: Foco

- [ ] 1-2 elementos têm cor de destaque; o resto está em neutro/cinza?
- [ ] Há hierarquia visual clara (título > dados > contexto > bordas)?
- [ ] Anotações guiam o olhar para o ponto principal?
- [ ] A informação mais importante está na posição de maior destaque?

## Princípio 5: Design

- [ ] Contraste suficiente entre texto e fundo?
- [ ] Funciona em preto e branco (impressão/fotocópia)?
- [ ] Não depende exclusivamente de cor (usa texto ou padrões também)?
- [ ] Fonte legível no formato final (min 12pt corpo, 18pt slides)?
- [ ] Consistente com os demais gráficos da apresentação?

## Princípio 6: História

- [ ] O título do gráfico conta a história? (não apenas "Gráfico 1")
- [ ] O gráfico tem um call-to-action implícito?
- [ ] Se alguém olhar só o título + gráfico, entende a mensagem?

## Teste Final: O Colega

Mostre o gráfico para um colega e pergunte:

1. **"O que você entendeu?"** (se não for sua mensagem, revise)
2. **"Para onde seu olho foi primeiro?"** (se não for o ponto principal, ajuste)
3. **"O que você gostaria de saber mais?"** (se houver lacunas, complete)
