# Checklist: Fluxo da Narrativa

> Use para verificar se sua apresentação conta uma história coesa.

## Estrutura Geral

- [ ] Início, meio e fim claramente identificáveis
- [ ] A história começa com o problema (não com os dados)
- [ ] A história termina com ação (não com "é isso")
- [ ] Mensagem principal repetida 3 vezes

## Conexão entre Slides

- [ ] Cada slide tem uma transição verbal preparada
- [ ] Não há pulos abruptos entre tópicos
- [ ] O slide N leva naturalmente ao slide N+1
- [ ] Se alguém perder um slide, ainda entende o contexto

## Clareza

- [ ] A Grande Ideia cabe em 1 frase
- [ ] A História de 3 Minutos está pronta e praticada
- [ ] Um colega entendeu a mensagem sem explicação extra
- [ ] O público sabe o que fazer após a apresentação

## Emoção e Engajamento

- [ ] A história tem um elemento de tensão ou surpresa
- [ ] Dados são usados para reforçar, não para substituir a narrativa
- [ ] Exemplos concretos ajudam a ilustrar pontos abstratos
- [ ] A apresentação tem ritmo (não é monótona)

## Call-to-Action

- [ ] Último slide: ação clara que o público deve tomar
- [ ] A ação é específica (não "pensem sobre isso")
- [ ] A ação é realista e alcançável
- [ ] Consequências são claras: o que acontece se agir vs não agir
