# Template: Preparação para Apresentação ao Vivo

> **"Não use seus slides como seu teleprompter!"** — Cole Knaflic

## Checklist de Preparação

### 1 Semana Antes
- [ ] Storyboard aprovado
- [ ] Slides criados (mínimo de texto)
- [ ] Gráficos finalizados
- [ ] História de 3 minutos escrita

### 3 Dias Antes
- [ ] Primeira apresentação simulada (para si mesmo, em voz alta)
- [ ] Ajustes nos slides baseados na simulação
- [ ] 3 perguntas difíceis identificadas
- [ ] Respostas preparadas para cada pergunta

### 1 Dia Antes
- [ ] Apresentação para um amigo/colega (simulação completa)
- [ ] Feedback recebido e incorporado
- [ ] Slides de apêndice preparados
- [ ] Backup dos slides (USB, nuvem, e-mail)

### 1 Hora Antes
- [ ] Equipamento testado (projetor, laptop, clicker)
- [ ] Fonte d'água por perto
- [ ] Respiração consciente (5 minutos)
- [ ] Aquecimento vocal (ler algo em voz alta)

## Estrutura de Anotações do Apresentador

Para cada slide, tenha suas anotações separadas:

| Slide | O que mostrar | O que dizer (palavras-chave) | Tempo |
|---|---|---|---|
| 1 | Título | "Bom dia. Neste trabalho..." | 30s |
| 2 | Contexto | "Dataset de 100k registros..." | 60s |
| 3 | Metodologia | "Split estratificado..." | 45s |
| 4 | RF | "RF com GridSearchCV..." | 60s |
| ... | ... | ... | ... |

## Dicas de Oratória

| Técnica | Descrição |
|---|---|
| **Pausa** | Faça uma pausa de 2-3 segundos antes do dado mais importante |
| **Contato visual** | Olhe para diferentes pessoas da plateia, não para o slide |
| **Voz** | Varie tom e volume. Monótono = sono |
| **Mãos** | Use gestos para reforçar (não cruze os braços) |
| **Postura** | Em pé, ereto, aberto. Não se apoie na mesa |
| **Velocidade** | Fale mais devagar do que você acha necessário |
| **Transições** | "Isso nos leva a..." "Mas o interessante é que..." |
| **Improviso** | Se algo der errado, respire, continue |

## O Que Fazer se...

| Problema | Solução |
|---|---|
| **Projetor quebrou** | Apresente sem slides (por isso você tem a História de 3 Minutos) |
| **Tempo reduzido pela metade** | Pule detalhes, vá direto para resultados + conclusão |
| **Pergunta que não sabe responder** | "Não sei, mas posso verificar" — anote e responda depois |
| **Plateia dispersa** | Faça uma pergunta, mude o tom, use uma pausa dramática |
| **Erro no gráfico** | "Este gráfico tem um erro. O dado correto é..." (honestidade > perfeição) |

## Pós-Apresentação

- [ ] Anote as perguntas que surgiram (para preparar melhor da próxima vez)
- [ ] Anote o que funcionou bem (para repetir)
- [ ] Anote o que não funcionou (para melhorar)
- [ ] Envie os slides + apêndice por e-mail (se combinado)
